ChemDraw Direct 2.0 Overview

ChemDraw Direct (CDD) is a lightweight JavaScript-based chemical drawing tool built using modern web technologies that can be easily integrated into web-based applications such as PerkinElmer Signals ™ Notebook via its published API.

What's New in This Release

ChemDraw Direct 2.0 is the most recent version of the application and it provides significant feature enhancements as described below.

Chemical Data Format Support

CDD 2.0 has been enhanced to have the same capability with ChemDraw Desktop to handle the following chemical data formats.

CDX
CDXML
MOLFILE 2K
MOLFILE 3K
SMILES

Compared to CDD 1.x, the data integrity is guaranteed in processing CDX/CDXML data. ChemDraw Web Service is not required any longer to load the data in CDX/MOLFILE/SMILES.

Rendering

CDD 1.x has limitations to render complex objects. CDD 2.0 has been enhanced to behave the same with ChemDraw Desktop in rendering.

Following are typical objects newly supported in CDD 2.0.

Complex text style or layout
Complex shapes, including all pre-defined templates
HELM data
Other objects which were not supported in CDD 1.x, such as tables, spectra, constraints, plates, plasmid maps, ellipsoid arcs, stoichiometry grids, bioshapes, and R-logic results

Besides the newly supported objects, CDD 2.0 has much better display quality on Retina (High-DPI) displays.

Nicknames

All the 518 nicknames pre-defined in ChemDraw Desktop are available in CDD 2.0. Operations on nicknames such as editing, rendering, expanding are supported. In CDD 1.x, only 169 nicknames were supported.

Drawing Tools

CDD 2.0 implemented more drawing tools which behave the same with ChemDraw Desktop.

Bond

All of the 20 bond tools which are available in ChemDraw Desktop are available in CDD 2.0. The following are the newly added bond tools in CDD 2.0.

Quadruple bond
Hashed bond
Wavy bond
Bold double bond
Dashed double bond
Double dashed double bond
Double either bond
Dative bond
Any bond
Double or Aromatic Bond
Single or Aromatic Bond
Single or Double Bond

Ring

All of the 10 ring tools which are available in ChemDraw Desktop are available in CDD 2.0. The following are the newly added ring tools in CDD 2.0.

Cycloheptane Ring
Cyclooctane Ring
Cyclopentadiene Ring

Arrow

All of the 56 arrow tools which are available in ChemDraw Desktop are available in CDD 2.0. In CDD 1.x, only five basic arrows were supported.

Templates

All templates that are pre-defined in ChemDraw Desktop are supported in CDD 2.0.

Extended Copy and Paste

In CDD 1.x, the back-end ChemDraw Web Service is required to support exchanging structures with external programs with copying and pasting. In CDD 2.0, the copying and pasting is done on client side only.

Limitations and Known Issues

GetCDXML action takes significant time to get the CDXML of complex structures. This issue appears only in Internet Explorer 11 browser
The cursor icon always shows a hand icon when a bond / ring tool is selected
The number in atom label changes to sub-script in the svg file generated using the getSVG option
No notifications appear on Dev Console while performing Group and Integral operations
Single bond attached to the Nickname "FRNAC" is not displayed properly in the ChemDraw Element
Structures with square brackets are not getting cleaned up properly when using the Clean Up tool
Structures with brackets are not getting cleaned up properly when using the Structure Clean Up tool
Stoichiometry table is not expanding vertically when multiple reactants / products are entered
Reagents are getting moved into the Reactant side when an RXN file is imported
Captions are missing after RXN file export, when it is above or below the reaction arrow
Reagent below the arrow is missing after RXN file export, when it is used before the slash symbol
The bond length between two isotopic atom labels get reduced when the bond is drawn in horizontal position
Once the text box is opened (editing mode), it does not close automatically unless any drawing action is taken. This can lead to data loss when the drawing is exported using any API command.
