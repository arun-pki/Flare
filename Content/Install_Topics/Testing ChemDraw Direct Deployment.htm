Verifying ChemDraw JS Deployment

The sample page of ChemDraw JS can be used to verify the deployment.

If ChemDraw JS is deployed using the installer, the sample page can be opened directly if the Show ChemDraw JS Sample Page option is checked at the last step of the installation wizard.

Or a menu item is available in Windows Startup menu to open ChemDraw JS Sample Page 17.1.0

Loading ChemDraw JS License on the Sample Page

The sample page also needs a valid ChemDraw JS license. A dialog will be shown on the sample page.

Please copy ChemDraw-JS-License.xml to the same folder of the sample page.

Installing a license for the sample page only allows sample page to run. The host application should still contain the code to load the license into ChemDraw JS when attaching. Please refer to the ChemDraw JS 17.1 Developers Guide for details.

After the license is loaded, the sample page should show the ChemDraw toolbar.

Testing ChemDraw JS Service

The tool Name to Structure can be used to test installation.

Click the Name to Structure tool button in toolbar.

Type "water" and click OK.

ChemDraw JS Service works as expected if H-O-H shows on the canvas
