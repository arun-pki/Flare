Tutorial 5: Templates

ChemDraw JS comes with a template library of pre-drawn structures and images. You can use a template to either start a new drawing or to modify one that already exists. In this tutorial, we use the phenanthrene template to create peroxydibenzene. The template and final drawing are shown below:

To add the template for phenanthrene:

In the toolbar, select the templates tool. The list of template palettes appears.
In the Aromatics palette, select the phenanthrene template.
Click in the canvas. Phenanthrene appears. Now you must remove two bonds from the center ring of the structure and add two oxygen atoms.

To modify phenanthrene:

In the toolbar, select the Eraser tool.
Click the bottom bond in the center ring.
Click the double bond at the top of the ring. It is now a single bond.
In the toolbar, select the Text tool.
Select one of the carbon atoms in the hydrocarbon chain (what was the middle ring).
Type o to change it to an oxygen atom.
Select the other carbon atom and change it to an oxygen. Peroxydibenzene is now shown.
