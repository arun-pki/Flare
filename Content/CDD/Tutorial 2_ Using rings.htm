Tutorial 2: Using Rings

This tutorial describes how to use rings to create the following structure:

 

Create the ring system:
Hold down the <Shift> key to change resonance structures when using the cyclopentadiene or benzene tools.
Using the Benzene tool, Shift-click in an empty area of the document.
Point to the center of the lower right bond in the benzene ring.
Click to fuse another ring.
In the final structure, there is only one double bond in the second ring. To correct the current structure, hover the mouse over one of the double bonds in the second ring and type 1. Do the same for the other double bond.

To create the double bond in the second ring, either hover the mouse over the bond and type 2 or click it.

Using the bond tool, click the carbon atoms circled in the figure below. Three new bonds will appear.

Hover your mouse over one of the bond shown below and type 2. Do the same for the other bond.

Hover your mouse over one of the bond shown below and type 2. Do the same for the other bond.

To connect the left terminal carbons, select the solid bond tool and click-drag a bond from one carbon to the other. The structure is complete.
