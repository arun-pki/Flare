Loading Chemical Data by Copy and Paste

The structures drawn on another chemical structure sketcher such as, ChemDraw Desktop, BIOVIA Draw, and Marvin Sketch can be copied and pasted to ChemDraw JS. Due to security policy of web browsers, ChemDraw Web Clipboard needs to be installed and enabled in the web browser to enable ChemDraw JS to have full access to the system clipboard. Otherwise, ChemDraw JS can only load the chemical data which is copied as CDXML text or other chemical text format.

Copying Chemical Data from ChemDraw Desktop without ChemDraw Web Clipboard

When ChemDraw Web Clipboard is not enabled, the gray clipboard icon is shown on the bottom-left corner of the canvas. The icon may not show at all if the extended copy and paste feature is not enabled.

1. Select the structures to copy in ChemDraw Desktop.
2. Select Edit > Copy As > CDXML Text in ChemDraw Desktop.
Note: Copying as CDXML Text can be done by pressing shortcut key <Ctrl>+D (<Command>+D on macOS).
3. Switch to ChemDraw JS canvas.
Note: You may need to click the canvas to let the web browser know that you are focusing on the ChemDraw JS canvas.
4. Click the Paste tool or press shortcut key <Ctrl>+V (<Command>+V on macOS) to paste the structure.

Copying and Pasting Chemical Data as Text without ChemDraw Web Clipboard

The chemical data in text format can be copied and pasted to ChemDraw JS without ChemDraw Web Clipboard. After pasted to ChemDraw JS, the chemical structure is rendered on ChemDraw JS canvas.

The following chemical text formats are supported:

CDXML
Molfile, Molfile v3000
SMILES

The following is an example of copying data as CDXML text:

1. Select the CDXML text to copy.
2. Switch to ChemDraw JS canvas.
Note: You may need to click the canvas to let the web browser know that you are focusing on the ChemDraw JS canvas.
3. Click the Paste tool or press shortcut key <Ctrl>+V (<Command>+V on macOS) to paste the structure.
Note: If Extended Copy and Paste is not activated, clicking the Paste tool will pop up a message dialog "These actions are unavailable via the tool" and nothing pastes onto the canvas.

Loading Chemical Data with ChemDraw Web Clipboard

With ChemDraw Web Clipboard enabled, ChemDraw JS has full access to the system clipboard. Hence, it can read the clipboard data put by desktop applications such as ChemDraw Desktop, BIOVIA Draw, and Marvin Sketch.

When ChemDraw Web Clipboard is enabled, the green clipboard icon is shown on the bottom-left corner of the canvas.

Following is an example of copying data from ChemDraw Desktop.

1. Select the structures to copy in ChemDraw Desktop.
2. Press shortcut key <Ctrl>+C (<Command>+C on macOS) to copy the selected data.
3. Switch to ChemDraw JS canvas and make sure the green clipboard icon shows.
Note: You may need to click the canvas to let the web browser know that you are focusing on the ChemDraw JS canvas.
4. Click the Paste tool or press shortcut key <Ctrl>+V (<Command>+V on macOS) to paste the structure.
Note: To enable ChemDraw Web Clipboard, please refer to Activating Extended Copy and Paste for details.
