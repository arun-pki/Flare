Shortcuts

Below is a list of key combinations for common tasks.
The shortcut keys described in this section are common to both Windows and Mac Operating Systems. However the only difference is that, if you use a MacOS, you need to use the <Command> key instead of the <Ctrl> key in the Key Combination.
