Loading Chemical Files by Drag and Drop

You can drag and drop an existing chemical file from your local computer to ChemDraw JS drawing area. When you drag the chemical file into the drawing area, its corresponding structure displays.

The following chemical formats are supported.

Extension Name Chemical Format
.cdx CDX
.cdxml CDXML
.mol Molfile, Molfile v3000
.smi SMILES
.smiles SMILES

To drag and drop a chemical file from your computer to ChemDraw JS:

Locate and select the CDXML file on your computer that you want to drag to ChemDraw JS.
Drag the CDXML file into the ChemDraw JS drawing area.

Release the mouse to drop the CDXML file into the drawing area. The structure displays as shown below:

If the drawing area contains existing structure, after the file is dropped into the drawing area, you are prompted to confirm if you want to replace the existing content or insert the new content to the drawing area.

When a CDX/CDXML file that contains style is loaded into ChemDraw JS, the loaded structure follows the merged style defined in the CDXML, and then new drawings will also follow this style.

When a CDX/CDXML file that contains style is pasted into ChemDraw JS, the pasted structure follows the current style of ChemDraw JS.
