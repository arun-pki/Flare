Copying Chemical Data and Pasting to External Applications

Copying and Pasting CDXML Text Data without ChemDraw Web Clipboard

Without ChemDraw Web Clipboard, ChemDraw JS can only copy the selected structure as CDXML text. The copied CDXML text can be pasted to another ChemDraw JS canvas directly.

To load the data to other applications, you can

Paste the CDXML text to a text file.
Change the extension name of the text file to .cdxml.
Open the file in other applications which support .cdxml, such as ChemDraw Desktop.

ChemDraw Desktop provides special paste option to avoid creating a new file.

Select Edit > Paste Special > MOL/CDXML Text in ChemDraw Desktop.
This paste special command can be done by pressing shortcut key <Alt>+<Shift>+<Ctrl>+P (<Alt>+<Shift>+<Command>+P on macOS).

Copying Chemical Data with ChemDraw Web Clipboard

With ChemDraw Web Clipboard enabled, ChemDraw JS has full access to the system clipboard. Hence, it can write the clipboard data to the system clipboard which can be recognized by desktop applications such as ChemDraw Desktop, BIOVIA Draw, and Marvin Sketch.

Following is an example of copying and pasting data to ChemDraw Desktop:

Select the structures to copy in ChemDraw JS.
Press shortcut key <Ctrl>+C (<Command>+C on macOS) to copy the selected data.
Switch to ChemDraw Desktop canvas.
Click the Paste tool or press shortcut key <Ctrl>+V (<Command>+V on macOS) to paste the structure.
To enable ChemDraw Web Clipboard, please refer to Activating Extended Copy and Paste for details.

Copying Structures to Microsoft Office Applications

With ChemDraw Web Clipboard enabled, you can copy your drawings from ChemDraw JS canvas and paste it into Microsoft Word, Excel or PowerPoint. Further, on Windows, if ChemDraw Desktop is installed on the machine, the pasted structures can be opened and edited in ChemDraw Desktop. ChemDraw JS also lets you copy the edited structure from the Microsoft Office application and paste it back to the ChemDraw JS canvas, making round-trip editing possible.

This feature requires ChemDraw Web Clipboard. To enable ChemDraw Web Clipboard, refer to Activating Extended Copy and Paste for details.
Round-trip editing may not work properly if an older version of ChemDraw Desktop is installed on the machine. Also, the round-trip editing may not work properly on other chemical structure sketchers such as BIOVIA Draw, and Marvin Sketch.

The following is an example of the steps required to perform round-trip editing using Microsoft Word:

Select the structures to copy in ChemDraw JS.
Press the shortcut key <Ctrl>+C (<Command>+C on macOS) to copy the selected data.
Switch to the Microsoft Word document.
You must ensure that ChemDraw Desktop is installed on the machine.
Press the shortcut key <Ctrl>+V (<Command>+V on macOS) to paste the structure.
Double-click the structure to open it in ChemDraw Desktop.
This works only on Windows operating system.
Make necessary edits, save and close ChemDraw Desktop. The structure in Microsoft Word is updated with your recent edits.
Press the shortcut key <Ctrl>+C (<Command>+C on macOS) in Microsoft Word to copy the structure.
Switch back to the ChemDraw JS canvas.
Press the shortcut key <Ctrl>+V (<Command>+V on macOS) to paste the structure.
