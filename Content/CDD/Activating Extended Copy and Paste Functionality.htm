Activating Extended Copy and Paste Functionality

Extended Copy and Paste functionality in ChemDraw JS enables you to use the keyboard shortcut keys to perform copying and pasting of structures in ChemDraw JS. You can press <Ctrl>+C (<Command>+C on macOS) on your keyboard to copy selected structures from ChemDraw Desktop or any popular chemical editor and press <Ctrl>+V (<Command>+V on macOS) to paste the copied structure into ChemDraw JS. Similarly, you can press <Ctrl>+C (<Command>+C on macOS) on your keyboard to copy selected structures from ChemDraw JS and press <Ctrl>+V (<Command>+V on macOS) to paste the copied structure into ChemDraw Desktop or any popular chemical editor. The copied structure can also be pasted into Microsoft Office applications such as Microsoft Word, and Microsoft PowerPoint.

To activate this feature on your browser, you have to allow ChemDraw JS to access the system clipboard, install and enable the extensions for your browser according to the instruction.

System Requirement

Supported Operating Systems

Windows 7 or later
macOS 10.12 or later

Supported Browsers

Google Chrome 53 or later
Mozilla Firefox 50 or later
Apple Safari 11 or later
Microsoft Internet Explorer 11
Note: Microsoft Edge will be supported in future.

In Microsoft Internet Explorer 11, the ChemDraw Web Clipboard applet should be enabled to support extended copy and paste. Java Runtime Environment(JRE) should be installed and enabled. If you use 32-bit and 64-bit browsers interchangeably, you will need to install both 32-bit and 64-bit Java in order to have the Java plug-in for both browsers. To check if a JRE is properly enabled, please open the page https://www.java.com/en/download/installed.jsp in IE to verify.

Allowing Access to the System Clipboard

ChemDraw JS has to get your permission to access the system clipboard. You will be prompted to grant permission when accessing ChemDraw JS for the first time.

To allow access to the system clipboard, click Allow. Otherwise, click Deny.

Withdrawing Permission

After you allow access to the system clipboard, you can withdraw permission anytime you want. To withdraw permission, click the clipboard icon on the bottom-left corner of the canvas. The activation dialog-box appears.

Un-check the Allow extension to run check-box and click OK.

Installing and Enabling ChemDraw Web Clipboard

To install and enable the ChemDraw Web clipboard

1. From the bottom-left corner of the drawing window, click the gray clipboard icon to view the detailed information and instruction to activate the feature.
Note: The instructions may differ based on your web browser or operating system.
Installing on Google Chrome or Mozilla Firefox 57+

The ChemDraw Web Clipboard application and the browser extension needs to installed.

Step 1. Installing ChemDraw Web Clipboard application

Installing on Windows
1. Click the download link provided in the first step of the instructions to download the installer.

2. Click Next to start the installation.
3. Accept the agreement and click Next.
4. Specify the location where the ChemDraw Web Clipboard should be installed and then click Next.
5. Specify the program's shortcuts in the Start Menu and click Next .
6. Review the installation summary and click Install.
7. Click Finish to complete the installation.
Installing on macOS
1. Click the download link provided in the first step of the instructions to download the installer.

2. Run the downloaded file. The Install ChemDraw Web Clipboard wizard appears.

3. Click Continue to start installation.
4. Specify how you want to install this software and click Continue.
5. Click Install.
6. Click Close to complete the installation.

Step 2. Installing the Browser Extension

Installing on Google Chrome
1. Click the download link provided in the second step of the instructions to install the extension.
2. Click the ADD TO CHROME button to install the extension.
3. Click the Add extension button to enable the extension.
4. ChemDraw Web Clipboard has been added to Chrome.
Note: Internet connection is required to install the extension for Chrome.
Installing on Mozilla Firefox 57+
1. Click the install link provided in the second step of the instructions to install the extension.
2. Click the Allow button.
3. Click the Install button.
4. ChemDraw Web Clipboard has been installed successfully.
Installing on Apple Safari 11+

The ChemDraw Web Clipboard application needs to be installed and the browser extension needs to be enabled.

Step 1. Installing the Application

Click the download link provided in the first step of the instructions to download the installer.

1. Click Agree to start the installation.
2. Drag ChemDraw Web Clipboard to your applications folder to install.
3. Close the installation window.
Step 2: Enabling the Extension
1. Find and launch the ChemDraw Web Clipboard application.
2. Click Open.
3. Click Show Safari Preferences to manage extensions.
4. Check ChemDraw Web Clipboard to enable the extension.
5. The Safari extension is enabled.
Installing on Internet Explorer 11

The ChemDraw Web Clipboard application needs to be installed and the Java plugin needs to be installed and enabled.

Step 1. Installing the Application
1. Click the download link provided in the first step of the instructions to download the installer.
2. Click Run to start the installation. The License Agreement window appears.
3. Accept the agreement and click Next. The Select Destination Location window appears.
4. Specify the location where the ChemDraw Web Clipboard should be installed and then click Next. The Select Start Menu Folder window appears.
5. Specify the program's shortcuts in the Start Menu and click Next. The Ready to Install window appears.
6. Review the installation summary and click Install.
7. Click Finish to complete the installation.
Step 2. Installing the Java plugin
1. Click the Click here link provided in the second step of the instructions to open the Java Downloads web page.
2. From the Java Downloads for All Operating Systems web page, follow the on-screen instructions to install a supported Java version.
3. After Java is installed, click the refresh link provided in the third step of the instructions to refresh the browser.
Step 3. Enabling Java plugin
After installing the Java plugin, you need to allow your browser to accept the Java plugin.

Click Allow to allow the applet running.
