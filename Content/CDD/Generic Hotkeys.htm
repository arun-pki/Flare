Generic Hotkeys (Tool switching hotkeys)

Generic hotkeys have been introduced to be present by default in ChemDraw JS without having to modify the hotkeys.xml file. When the mouse is not over an atom or the bond, you can press a key on the keyboard to switch between tools. The key to press is indicated between brackets in the tooltip while mousing over the toolbar. Those generic hotkeys are case sensitive, pressing <x> will for example select the single bond tool, while <Shift+x> will select the acyclic chain tool.
