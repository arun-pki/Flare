Loading and Getting Structure Data

Using the ChemDraw JS data loading APIs, the following file formats can be loaded into ChemDraw JS.

Chemical Format API
CDX(Base64 encoded) loadB64CDX
CDXML loadCDXML
Molfile (v2000, v3000) loadMOL
SMILES loadSMILES

End users can open local chemical files with supported file extensions by dragging and dropping to the ChemDraw JS canvas.

Extension Name Chemical Format
.cdx CDX
.cdxml CDXML
.mol Molfile, Molfile v3000
.smi, .smiles SMILES

Using the ChemDraw JS data getting APIs, ChemDraw JS can return chemical data of drawings in the following formats.

Chemical Format API
CDX (Base64 encoded) getB64CDX
CDXML getCDXML
Molfile getMOL
Molfile v3000 getMOLV3000
SMILES getSMILES
ISIS Reaction getRXN
ISIS Reaction v3000 getRXNV3000
InChI getInChI
InChIKey getInChIKey

Retrieving Structure Properties

getProperties
Returns a JSON object which list the properties of the current drawings.
This API requires the ChemDraw Web Service.
The example below shows the available properties of a benzene.
{
  "ID": "5",
  "NAME": "benzene",
  "FORMULA": "C<sub>6</sub>H<sub>6</sub>",
  "MW": "78.1140000006 g/mol",
  "EM": 78.0469501926,
  "INCHI": "InChI=1S/C6H6/c1-2-4-6-5-3-1/h1-6H",
  "SMILES": "C1=CC=CC=C1"
}

Exporting Images

ChemDraw JS allows the host application to export the current drawings to images using the following exposed APIs.
getImageUrl
getSVG
Limitation:ChemDraw Web Service is required for the getSVG API. Some chemical structures cannot be rendered correctly by SVG.
