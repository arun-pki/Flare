Configuring ChemDraw JS

ChemDraw JS allows the host application to configure the ChemDraw JS instances globally using single configuration file or each ChemDraw JS instance with separate configuration object or file.

Loading ChemDraw JS Configuration

By default, ChemDraw JS will load the configuration file named "configuration.json" from the same location with the library. The changes on this configuration file will affect all instances globally.
The default configuration can be overridden by setting the config or configUrl property of the AttachOptions in the API attach. The configuration set in the API attach only affects the instance being attached.
After a ChemDraw JS instance is created, the configuration can be updated by calling the APIs loadConfig or loadConfigUrl. The configuration set in the API loadConfig or loadConfigUrl only affects the current instance.
Please refer to the ChemDraw JS 17.1 API Reference Guide for more details.

Configuration Structure

The layout property manages the layout of the toolbar including the visibility of the tools.
Please refer to Customizing Toolbar Layout for details.

Features

The following features can be enabled or disabled in the configuration.
...
  "features": {
    "enabled": [
      "ShowWarnValence",
      "ShowWarnParens",
      "ShowWarnIsotopes",
      "ShowWarnStrayBonds",
      "ShowWarnStrayAtoms",
      "ShowWarnUndefinedStereo",
      "ShowWarnAmbigStereo",
      "ShowWarnStereoBtw",
      "ShowWarnLinearAtoms",
      "ShowWarnMisc",
      "ApplicationLogo",
      "WebService",
      "ExtendedCopyPaste"
    ],
    "disabled": []
  }
...
To enable or disable one feature, move the entry to the array of enabled or disabled property.

Chemical Errors Detection

The following chemical errors can be detected. Once an error type is detected on the chemical drawings, ChemDraw JS will show warnings to the end user.
Configuration Entry Chemical Error
ShowWarnValence Valence and Charge Errors
ShowWarnParens Unbalanced Parentheses
ShowWarnIsotopes Invalid Isotopes
ShowWarnStrayBonds Isolated Bonds
ShowWarnStrayAtoms Atoms Near Other Bonds
ShowWarnUndefinedStereo Unspecified Stereochemistry
ShowWarnAmbigStereo Ambiguous Stereochemistry
ShowWarnStereoBtw Stereobond between Chiral Atoms
ShowWarnLinearAtoms Linear Atoms
ShowWarnMisc Miscellaneous Warnings

Application Logo

By default, there is always a ChemDraw logo on the bottom-right of the ChemDraw JS workspace.
To hide this logo, move the entry "ApplicationLogo" to the disabled feature array.

Web Service

The following tools requires ChemDraw Web Service.

Structure to Name
Name to Structure

The following APIs requires ChemDraw Web Service

findReactions
getProperties
getSVG
getMolecularWeight
nameToStructure
structureToName
Once the web service feature is disabled, all the related tools are hidden and the related API fails. To enable the web service feature, the entry "WebService" should exist in the enabled feature array, and the chemservice property needs to be set with the correct URL to the end point of the REST service.

Extended Copy Paste

To exchange data with other desktop programs, ChemDraw JS has to access the system clipboard using the ChemDraw Web Clipboard.

If the extended copy paste feature is enabled, there is a clipboard icon on the bottom-left of the workspace to indicate the state of extension. If the feature is disabled, the icon does not appear.
