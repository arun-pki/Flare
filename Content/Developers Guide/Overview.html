Overview

ChemDraw JS modularized ChemDraw functionality for the web! Built on JavaScript and HTML5, ChemDraw JS can be plugged into web browsers and internal applications that need to be chemicalized. Inside its lightweight packages, ChemDraw JS carries the core functionality of the ChemDraw family including: your favorite drawing capabilities, advanced Name to Structure, Structure to Name, Hotkeys/Shortcuts, Structure Templates and Structure Query Tools. ChemDraw JS is used within ChemDraw Cloud, Signals Notebook Individual Edition, Lead Discovery and PerkinElmer Signals Lead Discovery to provide a seamless web-based chemical drawing experience.
PerkinElmer offers ChemDraw JS as a Service in the cloud so that you do not need to install and maintain your own ChemDraw JS service.
This guide is designed to instruct developers on the development of web-based applications that uses ChemDraw JS. It describes the procedures for loading and attaching it, as well as API usage and functionality.

Quick Start

Prerequisites

A valid software license is required to use ChemDraw JS. Contact PerkinElmer to get a valid license file. The file has to be placed to a location where your web application which embeds ChemDraw JS can access. Or the content of the license file can be hard-coded in your script, so that the license content can be passed to ChemDraw JS.

Starting with the Simplest Web Page with ChemDraw JS

The easiest way to get started is to start with the sample code provided below. In the sample code, the URL of the license file should be changed to the actual URL of the license file.

<!doctype html>
<html>
    <head>
        <title>
            Get Started with ChemDraw JS
        </title>
        <script
            src="https://chemdrawdirect.perkinelmer.cloud/js/chemdrawweb/chemdrawweb.js">
        </script>
    </head>
    <body>
        <div
            id="chemdrawjs-container"
            style="position: absolute; left:0; top: 0; width: 600px; height: 400px;">
        </div>
        <script>
            var chemdrawjsAttached = function (chemdraw) {
                // do something
            };

            var chemdrawjsFailed = function (error) {
                alert(error);
            };

            perkinelmer.ChemdrawWebManager.attach({
                id: 'chemdrawjs-container',
                callback: chemdrawjsAttached,
                errorCallback: chemdrawjsFailed,
                licenseUrl: 'https://your-own-url/ChemDraw-JS-License.xml'
            });
        </script>
    </body>
</html>

Save the modified sample code locally and open it with your browser. The ChemDraw JS toolbar and canvas is visible on the web page.

Refer to Creating and Attaching a ChemDraw JS Instance to understand how attaching works.

Invoking ChemDraw JS APIs

ChemDraw JS provides JavaScript APIs for host applications to interact with ChemDraw JS, for example loading existing chemical data to ChemDraw JS.

var chemdrawjsAttached = function (chemdraw) {
  chemdraw.loadSMILES('C1=CC=CC=C1', function (result, error) {
    if (error) {
      alert(error);
    }
  });
};

In the above code, the loadSMILES() API was invoked in the callback function of perkinelmer.ChemdrawWebManager.attach(). It loads the SMILES data to ChemDraw JS.
