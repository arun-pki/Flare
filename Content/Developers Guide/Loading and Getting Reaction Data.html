Loading and Getting Reaction Data

Loading Reaction Data

Similar to the loadCDXML API, the following APIs can be used to add products, reactants, and reagents.
addProduct
addReactant
addReagent
The fragments of the reaction are aligned automatically. A solid arrow and plus signs are added automatically.

Getting Stoichiometry Data of the Reaction

findReactions

It returns the following stoichiometry data:

Structure name
Formula
Molecule weight
InChI
SMILES
Example of data returned

Getting Information of Each Fragment in the Reaction

getAllFragmentsInfo
getSelectedFragmentsInfo
These two APIs return the information of all fragments or selected fragments.

For each fragment, it returns the following information:

Fragment ID in the document
The CDXML data
The thumbnail png image data encoded in Base64
The bounding (x, y, width, height in internal unit)

Labelling Reaction Fragments

labelFragments

labelFragments API is used to add labels (Roman numerals) under the reactant, reagents or products to number the structures.
