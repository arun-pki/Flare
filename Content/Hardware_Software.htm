Overview

ChemDraw JS modularized ChemDraw functionality for the web! Built on JavaScript and HTML5, ChemDraw JS can be plugged into web browsers and internal applications that need to be chemicalized. Inside its lightweight packages, ChemDraw JS carries the core functionality of the ChemDraw family including: your favorite drawing capabilities, advanced Name to Structure, Structure to Name, Hotkeys/Shortcuts, Structure Templates and Structure Query Tools. ChemDraw JS is used within ChemDraw Cloud, Signals Notebook Individual Edition, Lead Discovery and PerkinElmer Signals Lead Discovery to provide a seamless web-based chemical drawing experience.
This guide will list software requirements and hardware requirements for following three components:
ChemDraw JS JavaScript Library is the core JavaScript library to support most of the functionalities. It is the only required component which runs in end user's browser.
ChemDraw Web Clipboard is an optional component for end users who need extended copy and paste functionality to exchange data between ChemDraw JS and external applications. It needs to be installed and enabled separately on the end users' machine.
ChemDraw Web Service runs on the server. The end users don't need to install it locally.

Supported Operating Systems

Windows 7 or later (Microsoft .NET Framework 4.0 required)

Browser Exceptions

The following old versions of browsers are not supported.

Mozilla Firefox: All versions prior to 50
Apple Safari: All versions prior to 11

Microsoft Internet Explorer 11 requires Java Runtime Environment 1.7 or later.

Microsoft Edge is not supported.

Without the ChemDraw Web Clipboard, the host application which uses ChemDraw JS can still provide extended copy and paste functionality by implementing the required interfaces within the host application. This is necessary when the ChemDraw JS JavaScript Library is used in a web browser control. Please refer to ChemDraw JS 17.1 Developers Guide for details.
